# SV Type Coverage by Scenario

Reference guide showing which SV types are generated by each fungal scenario.

## Scenarios with All 5 SV Types ✓

These scenarios generate comprehensive coverage:

### 1. **te_rich_pathogen** - TE-Rich Pathogenic Fungi
- **SV Types**: INS, DEL, **DUP**, **INV**, **TRA** (all 5) ✓
- **Features**: High transposable element content
- **Real Example**: *Puccinia* (rust fungi) with Gypsy/Copia LTRs, RIP
- **Best for**: Testing TE-related SVs and full diversity
- **Use in**: `--scenario-set te_rich_pathogen`

### 2. **cross_phylum_hgt** - Cross-Phylum HGT Events
- **SV Types**: INS, DEL, **DUP**, **INV**, **TRA** (all 5) ✓
- **Features**: Horizontal gene transfer, GC shifts
- **Real Example**: *Batrachochytrium* (chytrid) with high HGT from bacteria
- **Best for**: Testing TRA and off-reference SVs
- **Use in**: `--scenario-set cross_phylum_hgt`

### 3. **cross_phylum_hgt_stress** - Extreme HGT Stress
- **SV Types**: INS, DEL, **DUP**, **INV**, **TRA** (all 5) ✓
- **Features**: Extreme HGT, GC deviation
- **Real Example**: Extreme evolutionary stress scenario
- **Best for**: Edge case testing
- **Use in**: `--scenario-set cross_phylum_hgt_stress`

## Partial Coverage Scenarios

### 4. **pathogenic** - Plant Pathogenic Fungi
- **SV Types**: DEL, INS, **TRA** (3/5)
- **Missing**: DUP, INV
- **Features**: Fungal plant pathogens
- **Real Example**: *Fusarium*, *Magnaporthe*

### 5. **two_speed_pathogen** - Two-Speed Evolution
- **SV Types**: DEL, INS, **INV** (3/5)
- **Missing**: DUP, TRA
- **Features**: Fast-evolving effector regions + conserved regions
- **Real Example**: *Leptosphaeria*, *Colletotrichum*

### 6. **rust_smut_te_heavy** - Rust/Smut Fungi
- **SV Types**: DEL, INS, **TRA** (3/5)
- **Missing**: DUP, INV
- **Features**: Rust/smut-specific TE patterns
- **Real Example**: *Puccinia*, *Ustilago*

### 7. **hgt_receiver** - High HGT Recipient
- **SV Types**: DEL, INS, **TRA** (3/5)
- **Missing**: DUP, INV
- **Features**: High horizontal gene transfer
- **Real Example**: *Batrachochytrium dendrobatidis*

### 8. **arbuscular_mf** - Arbuscular Mycorrhizal Fungi
- **SV Types**: DEL, INS, **DUP** (3/5)
- **Missing**: TRA, INV
- **Features**: Asexual/clonal (no TRA), high repeats
- **Real Example**: *Rhizophagus irregularis*

### 9. **giant_amf** - Giant AMF
- **SV Types**: DEL, INS, **DUP** (3/5)
- **Missing**: TRA, INV
- **Features**: Very large genome (150 Mb), repeat-rich
- **Real Example**: *Rhizophagus* species

## SV Type Definitions

- **INS** (Insertion): Sequence added to genome
- **DEL** (Deletion): Sequence removed from genome
- **DUP** (Duplication): Segment copied within genome
- **INV** (Inversion): Segment reversed/inverted
- **TRA** (Translocation): Segment moved between contigs
- **OFF_REF** (Off-reference): Novel sequence not in reference

All detected by MycoSV across 3-layer hierarchical routing architecture.
