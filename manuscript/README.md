# MycoSV manuscript artifacts

Drop-in artifacts for the MycoSV manuscript draft: figures (PNG + SVG),
tables (Markdown + TSV), raw aggregated data, and a key-messages overview.

## Start here

- **[KEY_MESSAGES.md](KEY_MESSAGES.md)** — the headline claims, each anchored
  to a specific file and number. Read this first.

## Layout

```
manuscript/
├── KEY_MESSAGES.md                  # take-home messages with file anchors
├── README.md                        # this file
├── figures/                         # PNG (HTML preview) + SVG (Illustrator)
│   ├── fig1a_panel_read_validation_rate.{png,svg}
│   └── fig1b_panel_pangenome_lift.{png,svg}
├── tables/                          # Markdown (paste-ready) + TSV (machine)
│   ├── table_method_comparison_pangenome_vs_single_ref.{md,tsv}   # HEADLINE
│   └── table1_pangenome_vs_single_reference.{md,tsv}              # per-sample
└── data/                            # raw panel-fold aggregates
    ├── panel_read_validation_rate.tsv
    └── panel_pangenome_lift.tsv
```

## Regenerate

Everything in this folder is regenerated by one command from the project
root:

```
.venv/bin/python plot_mycosv_pangenome_calls.py \
  --panel-dir experiments/million_real/full_fungal_assembly_20260522_070100/assembly/by_query \
  --outdir manuscript/_regen
```

(Reorganize the `_regen/` output into `figures/`, `tables/`, `data/` to
match this layout.)
